# The Programming Language  zlang

## Features
- Purely functional
- Type inference by Hindley-Milner
- Type support for integer, boolean, string, list and tuples
- Easy, simple syntax

## Usage

```
$ ./zlang
Commandline interpreter for zlang
zlang>
```

## Getting started

```
$ ./zlang
Commandline interpreter for zlang
zlang>:string on
zlang>"Hello World!"
Hello, World!
zlang>
```

## Unit Tests

```
$ make tests
```
